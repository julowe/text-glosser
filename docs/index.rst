Text Glosser Documentation
==========================

Welcome to the Text Glosser documentation!

Text Glosser is a comprehensive text analysis and glossing application that provides 
word-by-word definitions and grammatical information using multiple linguistic 
dictionaries and resources.

Features
--------

* Multi-dictionary support for analyzing text
* CLI and Web UI interfaces
* Multiple output formats (Markdown, JSON, CoNLL-U)
* Session management with configurable data retention
* Built-in dictionaries for Sanskrit, Arabic, and Chinese
* Extensible architecture for adding custom dictionaries
